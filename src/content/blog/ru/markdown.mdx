## Заголовок H2 (Основной раздел)

Это вводный абзац текста-рыбы. Мы проверяем, как Astro обрабатывает базовые стили. Лорем ипсум долор сит амет, консектетур адиписцинг элит. Донек а либеро фелис.

### Заголовок H3 (Подраздел)

Абзацы разделяются пустой строкой. Вот пример **жирного текста** и *курсива*. Можно совмещать: ***жирный курсив***. Также поддерживается ~~зачеркнутый текст~~.

## Код и подсветка синтаксиса

Встроенный код выглядит `вот так`. А ниже представлен многострочный блок кода с подсветкой синтаксиса (Astro использует Shiki или Prism из коробки):

```javascript
// Тестовый JavaScript код
const astro = {
  type: 'framework',
  speed: 'blazing fast',
  greet() {
    console.log(`Welcome to ${this.type}!`);
  }
};

astro.greet();
```

Пример кода на HTML:

```html
<div class="card">
  <h2>Заголовок карточки</h2>
  <p>Описание контента.</p>
</div>
```
